HTML: Tags e Atributos

HTML e suas Tags

O HTML é uma linguagem de marcação interpretada pelo navegador. Esta linguagem utiliza tags ou elementos específicos para dizer ao navegador como as coisas devem ser interpretadas. Vamos dar uma olhada em alguns conceitos e sintaxe para declararmos as tags, ou seja, elementos HTML.

As tags são colocadas dentro de sinais de menor que < e maior que >. Perceba que a tag de fechamento leva o caractere /. As tags são quase sempre seguidas de tags de fechamento correspondentes.

Os exemplos a seguir são puramente didáticos, portanto não tags reais. Elas servirão para entendermos como o HTML é estruturado.

<tagDeAbertura>Conteúdo</tagDeFechamento>

As tags também podem ser aninhadas umas dentro das outras.

<pizza>
  <sabor>Queijo</sabor>
  <sabor>Calabresa</sabor>
  <sabor>Cebola</sabor>
</pizza>

As tags utilizam atributos para adicionar funcionalidades ou descrever comportamentos.

<tagDeAbertura atributo="valorDoAtributo">Conteúdo</tagDeFechamento>

Algumas tags possuem fechamento próprio e normalmente não envolvem algum conteúdo. A tag <img> é um exemplo que contém fechamento próprio. Como você pode ver, ainda há uma / no final da tag. Observe também o uso dos atributos. 

 <img src="pastaRaiz/arquivo.jpg" alt="descrição para acessibilidade." />

Tags comuns

Aqui você verá algumas das tags HTML mais comuns ao criar aplicações web. Esta será uma lição interativa, então poderemos ver o resultado do uso dessas tags em um navegador. Vamos construir uma página de viagens.

Títulos

As tags HTML <h1> até <h6> representam seis níveis de elementos de bloco para títulos. O <h1> é o nível mais elevado, e o <h6> é o mais baixo. É sempre importante respeitar a hierarquia de títulos.

Parágrafos

A tag HTML <p> representa um parágrafo com texto e é um elemento de bloco.

Divs

O elemento HTML <div> é um recipiente genérico para fluxo de conteúdo. Nós não utilizaremos neste momento, mas à medida que avançarmos em nossa grade. É uma tag principalmente utilizada para agrupar elementos com o objetivo de estilização e posicionamento.

A <div> é um elemento de bloco. Os navegadores normalmente exibem o elemento de nível de bloco colocando uma nova linha antes e depois do elemento.

<div>
  <p>Qualquer tipo de conteúdo aqui.</p>
</div>

Spans

O elemento HTML <span> é um elemento de linha principalmente para fins de estilo.

A tag <span> ocupa apenas o espaço delimitado pelas tags que o definem. Basicamente isso significa que você pode colocá-lo em volta de uma palavra ou algo semelhante.

<p>Qualquer tipo de conteúdo <span>aqui</span>.</p>

Conclusão
