Aula: Introdução ao HTML

HTML - Linguagem de Marcação de Hipertexto

O HTML é uma linguagem de marcação que fornece um modo estruturado de criar páginas web.

Ao utilizarmos as tags que o HTML nos disponibiliza, podemos controlar como quase tudo é apresentado no navegador de um usuário. É o navegador que interpreta as tags para apresentar o que o usuário final verá na tela.

Estrutura básica de páginas HTML

Nesta seção daremos uma olhada na estrutura de um arquivo HTML. Vamos discutir as tags básicas necessárias para ter uma página bem formada para exibição na internet. A partir de agora, entraremos no contexto do HTML5.

Estrutura básica de um arquivo HTML

<!DOCTYPE html>
<html>
  <head>
    <title></title>
  </head>    
  <body>
  </body>
</html>

DOCTYPE

<!DOCTYPE html> informa ao navegador qual versão do HTML será usada neste documento. Neste caso, estamos declarando o HTML5.

DOCTYPE é uma declaração, portanto não é considerado como uma tag. O que você realmente precisa entender sobre esta declaração é que ela deve estar presente na primeira linha do seu arquivo HTML.

HTML

<html> é a tag que representa o elemento raiz (elemento de nível superior) de um documento HTML, por isso ela também é chamada de elemento raiz. Todos os outros elementos devem ser descendentes deste elemento. Observe a tag de fechamento </html> na última linha.

Head

A tag <head> contém informações descritivas sobre a sua página, como o título da aba, por exemplo. É aqui onde você insere configurações como meta e arquivos CSS para estilização. Observe que a tag <title> possui uma tag de fechamento </title> assim como a tag </html>.

Title

A tag <title> define o título do documento que é exibido na barra superior do navegador, ou seja, na aba da página. Esta tag pode conter apenas texto e qualquer outra tag colocada dentro dela é ignorada. Além disso, é seguida da tag de fechamento </title>.

Importante!

Todas as tags acima são configurações para o navegador, a partir de agora vamos observar tags que irão reproduzir um efeito visual dentro do documento.

Body

A tag <body> representa o conteúdo de um documento HTML. Esté é um elemento único em um documento e representa a parte visual de uma página web. Ela também possui o fechamento </body> 

<html> 
  <head>
    Configurações da página.
  </head>
  <body>
    Parte visual.
  </body>
</html>

Diferenças entre configurações do navegador e o que o usuário pode ver

Quando falamos em configurações do browser, estamos nos referindo a compatibilidade, título da página e etc, mas no momento não vamos focar muito nessa parte.

E quando falamos em parte visual, estamos nos referindo a tudo o que aparece para o usuário, por exemplo, uma página de login ou esse próprio site que você está acessando para ler e assistir esta aula.

Conclusão

Nesta aula vimos as principais tags e como utilizá-las. O HTML possui muitas tags, e ao longo do curso, trabalharemos com as principais e mais habitualmente vistas em páginas web.
