Editor de Texto: Instalando e configurando o Visual Studio Code

O que é um editor de texto?

O editor de código ou editor de código-fonte é um software projetado especificamente para ajudar os desenvolvedores a programar. Na prática, eles são editores de texto com algumas funcionalidades adicionais para gerenciar e editar código.

Visual Studio Code

Um dos editores mais completos disponíveis no mercado, é multiplataforma (Windows, Mac OS, Linux) e um editor de código grátis.

Instalando o Visual Studio Code

Para a instalação você precisará seguir os seguintes passos:

  1. Abra o navegador;
  2. Abra a opção do fabricante através deste link;
  3. Selecione a plataforma de seu Sistema Operacional;
  4. Espere a conclusão do download;
  5. Clique no arquivo de instalação baixado. Caso não esteja na barra inferior de seu navegador, você poderá encontrá-lo na pasta downloads de seu computador;
  6. Concorde com os termos e clique para avançar;
  7. Após alguns passos, surgirão algumas sugestões para você. Deixe como está e marque a opção de criar um atalho na área de trabalho. Após isso, clique em avançar;
  8. Agora você só vai precisar concluir a instalação e inicar o Visual Studio Code.

Instalando Extensões

O Visual Studio Code oferece uma variedade enorme de extensões. Hoje instalaremos a extensão Live Server por meio deste passo a passo:

  1. Com seu Visual Studio Code aberto, na barra lateral esquerda, procure por extensions;
  2. Nesta página haverá uma lista de extensões;
  3. Procure por Live Server na barra de pesquisa;
  4. Clique em install.
Devo utilizar a extensão Live Server?

Normalmente, quando você quer fazer uma alteração no seu código ou escrever algo novo, é preciso atualizar a página manualmente para ver as alterações. Em outras palavras, se você fizer 100 alterações no seu código a cada dia, precisará atualizar o navegador 100 vezes. Com o Live server, você automatiza este processo deixando-o dinâmico.

Outras extensões interessantes

Além do Live Server, o Visual Studio Code disponibiliza uma série de extensões que podem te ajudar a ganhar tempo. Aqui estão algumas:

  • Auto Rename Tag

    - Quando estamos trabalhando com HTML e precisamos renomear tags de abertura e fechamento. Com esta extensão basta renomear apenas uma delas.

  • Color Highlight

    - O Color Highlight é uma ótima extensão para o Front-end. Ela identifica quando você escreve o código de uma cor e a exibe no editor.

  • Material Icon Theme

    - Dê uma cara nova nos seus ícones com esta extensão. Além de deixar os ícones mais bonitos, ajuda a localizar visualmente cada arquivo.

  • Auto Complete Tag

    - Ajuda a completar automaticamente a escrita de uma tag HTML.

Dicas para aumentar a produtividade no Editor de Código

Abertura Rápida

Este atalho ajuda a abrir arquivos rapidamente. Depois disso, use as setas do teclado para encontrar o arquivo. Depois de selecionar, toque na tecla de seta para a direita.

  • Windows: CTRL+P
  • Mac: CMD+P

Paleta de Comandos

Neste atalho você tem acesso à lista de comandos. Ao invés de ir de menu em menu, você pode utilizar o atalho e digitar uma palavra-chave relacionada ao comando que está procurando.

  • Windows: CTRL+SHIFT+P
  • Mac: CMD+SHIFT+P

Cursor de Seleção Múltipla

O cursor de seleção múltipla é uma ótima ferramenta para selecionar com precisão vários itens iguais ao mesmo tempo.

  • Windows: CTRL+D
  • Mac: CMD+D

Alternar a barra lateral

Usar a largura total do editor pode ser útil às vezes. Para casos como esses, a barra lateral de alternância é uma ótima solução.

  • Windows: CTRL+B
  • Mac: CMD+B

Copiar Linha

É comum entre os desenvolvedores e uma das dicas e truques mais eficazes. Por esse motivo, ter um atalho para ajudar a simplificar esse processo é fundamental.

  • Windows: SHIFT+ALT+UP ou SHIFT+ALT+DOWN
  • Mac: OPT+SHIFT+UP ou OPT+SHIFT+DOWN

Bloco de código de comentário

Comentar blocos de código é necessário quando você está modificando código ou rastreando um bug e, por ser tão comum, os atalhos podem ser extremamente úteis.

  • Windows: SHIFT+ALT+A (comentário de várias linhas), CTRL+K+C (comentário de linha única)
  • Mac: SHIFT+OPT+A

Voltar/avançar

Mova-se rapidamente entre um local e outro usando este atalho prático.

  • Windows: ALT+ —> ou + –>
  • Mac: ^-
