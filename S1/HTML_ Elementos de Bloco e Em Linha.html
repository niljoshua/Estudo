HTML: Elementos de Bloco e Em Linha

Introdução

Os elementos do HTML podem ser separados em dois grandes grupos, os elementos de bloco e os elementos em linha. Estes são conceitos muito importantes.

Futuramente, com o CSS, vocês vão conseguir mudar o comportamento padrão destes elementos. Por hora, vamos entender o que é e como funciona esta classificação.

Elementos em linha ou inline

Os elementos em linha ocupam apenas o valor necessário da largura para que eles apareçam na tela. São chamados de elementos em linha por poderem dividir a mesma "linha" com outros elementos, caso seja possível.

Exemplos de elementos em linha

<span></span> : Não possui valor semântico, é simplesmente uma tag genérica.

<strong></strong> : Tag semântica para indicar importância no texto.

<em></em> : Tag semântica para indicar ênfase no texto.

<a></a> : Tag âncora para criar hiperligação.

<img> : Tag para inserir uma imagem.

Elementos de bloco ou blocks

Os elementos de bloco ocupam todo o espaço horizontal disponível e iniciam uma nova linha do documento. Os novos elementos são inseridos um abaixo do outro, sempre em uma nova linha.

Exemplos de elementos de bloco

<div></div> : Não possui valor semântico, é simplesmente uma tag genérica.

<h1></h1> : Tag de título.

<p></p> : Tag de parágrafo.

<ul><ul> : Tag de lista desordenada.

<ol></ol> : Tag de lista ordenada.

<form></form> : Tag de formulário.

Block X Inline

Agora que aprendemos sobre elementos de bloco e elementos em linha, vamos ver a diferença entre eles na prática.

  • Vamos utilizar uma <section> para envolver nossos elementos.
  • Vamos criar dois elementos em linha: utilizaremos dois <span>, com o texto "Hello World" em cada um deles.
  • Vamos criar dois elementos de bloco: utilizaremos duas <div>, e vamos inserir em cada <div> um parágrafo com o texto "HTML".

Resultado no navegador

Hello World Hello World

Hello World

Hello World

Resultado no navegador

<section>
  <span>Hello World</span>
  <span>Hello World</span>
  <div>
    <p>Hello World</p>   
  </div>
  <div>
    <p>Hello World</p>   
  </div>
</section>

Vale a pena ressaltar que quando utilizamos elementos de linha em seguida, eles ficarão lado a lado, porém elementos de bloco ficarão na linha seguinte.

Isso acontece porque cada elemento de bloco ocupa a largura total disponível, enquanto o elemento de linha ocupa somente a largura necessária para que ele apareça em tela.

Conclusão

Agora é com você: tente reproduzir os conceitos de elementos inline e block. Lembre-se, a prática é extremamente importante para tornar seu conhecimento mais sólido.
