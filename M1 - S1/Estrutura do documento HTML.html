Estrutura do documento HTML

Introdução

A estrutura base do HTML vai se tornar algo muito comum no seu dia-a-dia. Nesta aula vamos revisar a estrutura base para que você entenda o significado de cada elemento.

<DOCTYPE html>

É uma regra/declaração do HTML que tem o intuito de anunciar ao navegador o tipo/extensão do arquivo.

Ele garante que o navegador siga as especificações do HTML, evitando renderizar algo incompatível com este tipo do documento.

<html></html>

Esta tag representa a raiz do documento, e todas as outras tags são descendentes dela.

<head></head>

Esta tag é responsável por armazenar as configurações, ou seja, metadados da página e título da aba.

<meta/>

Esta tag tem a funcionalidade de representar elementos que não podem ser representados por outras tags.

Um dos metadados fornecido por esta tag e que usamos bastante é o charset, que define o conjunto de caracteres que utilizaremos em nossa página. Usamos o `utf-8`, mas existem outros.

A tag <meta>, assim como toda e qualquer tag de metadado, é inserida na tag <head>.

<body></body>

Esta é a tag que envolve todo o conteúdo que aparece no navegador, ou seja, títulos, parágrafos, listas, âncoras e muita outras tags HTML.

Conclusão

Esta aula teve como objetivo explicar a função de algumas tags vistas até agora. Se precisar, revisite esta aula para rever o conteúdo.
