Aula: Incorporando imagens

Introdução

As imagens são inseridas em nosso documento HTML através da tag <img>. Este elemento, ou tag, possui fechamento automático.

Anatomia e atributos

Existem vários atributos para uma tag <img>. Vamos aprender um pouco sobre os principais e como declará-los.

src

Este atributo representa o caminho da imagem que você está inserindo no seu documento e é obrigatório para o elemento <img>.

alt

Este atributo define o texto alternativo da imagem. Os usuários têm acesso a este texto caso a imagem não esteja em um dos formatos suportados ou não consiga ser renderizada na página.

Apesar de não ser obrigatório, é essencial para tornar sua imagem adequada para fins de acessibilidade. Quando usamos este atributo, damos a opção de um leitor de tela interpretar em forma de texto a imagem inserida.

height

Este atributo define a altura do elemento. Quando usado na tag <img> adotamos por padrão um valor em pixels.

width

Este atributo define a largura do elemento. Quando usado na tag <img> adotamos por padrão um valor em pixels.

Exemplos 

// Imagem usada como link
<a href="https://developer.mozilla.org/pt-BR/" target="_blank">
  <img src="https://mdn.mozillademos.org/files/6851/mdn_logo.png"/>
</a>

// Imagem com width e height fornecendo largura e altura
<img src="some_image.png" width="200" height="200"/>

// Imagem com alt fornecendo uma descrição.
<img src="some_image.png" alt="MDN"/>

No exemplo acima, a tag <img> foi utilizada em conjunto com a tag <a> para fazer com que a imagem se torne um elemento clicável. Assim, é possível redirecionar o usuário para outra página ou seção.

Organizando imagens

Após aprender como utilizar imagens no HTML, devemos organizar as imagens no projeto. Abaixo listaremos algumas dicas importantes.

Primeira dica

Crie uma pasta img em seu projeto e deixe todas as imagens que vai usar dentro desta pasta.

Atenção

Ao utilizar imagens que estão armazenadas em nosso computador, conseguimos ter um resultado local. Também é possível utilizar imagens que já estão hospedadas na internet.

É importante criar uma pasta img dentro do projeto, desta forma a imagem vai estar vinculada ao projeto.

Segunda dica

Não é uma boa prática usar link de imagens que estão fora de seus domínios. Usar imagens de links externos pode ser perigoso, pois caso o site saia do ar ou por algum motivo deixe de existir, ou até mesmo se a imagem mudar, sua aplicação vai perder a referência da imagem e não conseguirá renderizá-la.

Esteja sempre atento a sites que disponibilizam o download ou domínio de imagens sem direitos autorais.

Terceira dica

Se você criou a pasta img como sugerido, o caminho para sua imagem será sempre este: 

"./img/nomedaimagem.extensão"

Conclusão

A organização e nomenclaturas de pastas não é uma regra, mas por enquanto aconselhamos que faça desta forma. Futuramente você vai aprender sobre organização de projetos. Nos projetos que desenvolverá adiante no curso, crie o arquivo index.html na raiz do seu projeto.
