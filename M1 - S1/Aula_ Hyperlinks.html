Aula: Hyperlinks

Introdução

Os links, ou hyperlinks, são representados pela tag de âncora <a>. Esta um hiperlink que aponta para um documento inteiro ou para um elemento específico dentro de um documento.

Um hiperlink pode levar o usuário para outras páginas da internet, locais dentro da mesma página, arquivos, endereços de e-mail, ou qualquer outra URL. O link clicável é o texto ou elemento inserido entre as tags <a></a>.

Anatomia básica da tag de âncora

Existem vários atributos para uma tag âncora. Vamos ver alguns dos mais comuns.

href

É a URL ou o destino para onde o hiperlink aponta.

target

Indica onde exibir a URL do link. Abaixo estão os valores mais utilizados:

  • _self : Carrega a URL na mesma janela atual do navegador. Este é o comportamento padrão (em inglês default) se o atributo target não for especificado.
  • _blank : Carrega a URL numa nova janela ou aba do navegador, de acordo com a configuração atual do navegador.

Exemplos básicos de âncora

// Abre em uma nova janela ou aba 
<a href="http://www.google.com" target="_blank">google.com</a>

// Abre na mesma janela
<a href="http://www.google.com" >google.com</a>

// Abre o cliente de e-mail padrão do usuário
<a href="mailto:user@example.com" >Email User</a>

// Leva a outro arquivo local no mesmo site
<a href="#page-section" >Link to page section</a>

// Este link externo abre em uma nova janela ou aba 
// Aqui utilizamos uma imagem ao invés de texto
<a href="https://developer.mozilla.org/pt-BR/" target="_blank">
  <img src="https://mdn.mozillademos.org/files/6851/mdn_logo.png" />
</a>

Conclusão

É importante conhecer a anatomia das tags que você utiliza, e para isso a documentção é a melhor maneira de encontrar exemplos de usabilidade e explicações completas sobre o elemento procurado.
