Aula: Tags semânticas

Introdução

Quando vamos criar um site, é necessário pensar em vários fatores, como estrutura semântica, acessibilidade e motores de busca.

Tags semânticas possuem significado

Quando falamos em interpretar o código, nos referimos ao significado. Alguns programas e motores de busca irão ler o nosso site e identificar os tipos de conteúdos ali existentes, vendo quais são mais importantes, onde fica o rodapé, onde fica o menu, entre outros.

Acessibilidade

Não só os robôs do Google buscam identificar onde se localiza cada conteúdo do HTML. Aplicativos utilizados por pessoas com problemas de visão, por exemplo, procuram identificar as seções e a importância de cada conteúdo para interpretar e fornecer o áudio para o usuário.

Estrutura de página divida em blocos

Quando acessamos um site, temos uma divisão dos conteúdos, como seção com fotos, menu, rodapé, e etc. Na imagem abaixo mostraremos a estrutura básica de um projeto.

Mas, pensando em estrutura HTML, como podemos fazer a divisão desse conteúdo?

Podemos separar o conteúdo utilizando a tag <div>, como no exemplo abaixo:

Código da imagem acima

<body>
  <!-- Menu -->    
  <div> 
    <a href="endereço">Início</a>
    <a href="endereço">Sobre</a>
    <a href="endereço">Contato</a>
  </div>
  <!-- Conteúdo Principal -->  
  <div> 
    <h1>Texto principal da página</h1>
    <p>Texto de descrição</p>
    <a href="endereço">Link</a>
  </div>
</body>

Porém, esta é uma divisão apenas para fins estéticos, pois a tag <div> não possui significado, ou seja, não possui semântica.

HTML 5

Na versão 5 do HTML, temos a possibilidade de deixá-lo mais descritivo, tanto para quem escreve ou para quem lê o código, quanto para os motores de busca (Google) e aplicativos de acessibilidade.

Para isso, vamos utilizar tags com nomes mais descritivos.

Podemos englobar os conteúdos que representam as seções do nosso website da seguinte forma: 

<section>
  <!-- Aqui fica o conteúdo -->
</section>

Sabe aquele conteúdo que fica em evidência bem no início da página? Como uma capa de perfil, capa de uma notícia ou post, ou um slogan de uma empresa.

Podemos envolver esse tipo de conteúdo utilizando a tag <header> da seguinte forma: 

<header>
  <!-- Aqui fica o conteúdo -->
</header>

Determinando conteúdo principal da página

Podemos utilizar as tags semânticas para determinar qual conteúdo da página deve ter mais importância.

Para isso, temos a tag <main>, envolvendo os elementos que terão uma importância maior naquela página.

No nosso código, faremos da seguinte forma:

<main>
  <!-- Aqui fica o conteúdo principal -->
</main>

Artigos, Notícias e Posts

Quando estamos criando um blog, podemos desenvolver o HTML de maneira que seja possível identificar que aquele trecho específico será um artigo.

Para conteúdos relacionados à artigos, notícias e posts temos uma tag especial: a tag <article>. Ela será muito utilizada por você durante o primeiro módulo do curso.

Conclusão

Entender sobre semântica é fundamental para desenvolvedores Front-End. Mecanismos de busca dão preferência e maior importância para aplicações que possuam tags semânticas afim de diferenciar a importância e significado.
