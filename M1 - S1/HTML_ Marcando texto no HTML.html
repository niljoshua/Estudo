HTML: Marcando texto no HTML

Introdução

Você já sabe que o HTML é uma linguagem de marcação de texto, mas como devemos marcá-los? Nesta aula vamos falar um pouco sobre as tags de marcação.

A primeira coisa na qual devemos pensar é a necessidade e/ou importância de cada elemento em cada trecho do texto que estamos produzindo.

Quando estamos criando um texto, usamos recursos da linguística para dar alguns toques visuais. Ao considerarmos uma palavra como importante é comum deixarmos ela com um estilo diferente, portanto utilizamos negrito, itálico ou sublinhado. Usamos também recursos de títulos e subtítulos em nossas escritas manuais.

Agora que você já entende a importância dos elementos textuais, vamos aprender a aplicar estes conceitos no HTML.

Títulos

No HTML temos as tags de marcação de títulos com vários níveis de relevância.

Esta hierarquia varia entre h1 até h6.

  • Este é o H1

  • Este é o H2

  • Este é o H3

  • Este é o H4

  • Este é o H5
  • Este é o H6

Formas de uso

As tags de título precisam de abertura e fechamento.

<h1>Sou um título</h1>

Vamos avaliar a anatomia da construção do trecho da marcação.

  • Primeiro abrimos a tag de título <h1>
  • Fechamos a tag <h1>
  • Entre a abertura e o fechamento inserimos o conteúdo do título

É desta maneira que marcamos nossos títulos no HTML, mas um texto não é constituído somente de títulos, portanto é necessário desenvolver o texto fora do escopo de título.

Marcando o texto com Parágrafos

Nossa primeira tag para marcação de texto é a tag <p> referente aos parágrafos. Novamente o sentido do uso desta tag é semelhante ao da linguística.

Um texto é constituído de início, meio e fim e você deve utilizar alguns recursos para que isso fique claro para quem está lendo, por isso é comum quebrarmos nossos textos em parágrafos.

Tente não deixar textos soltos em seu documento, afinal isto é uma péssima prática. Sempre utilize as tags de marcação para envolvê-los.

A tag referente ao parágrafo não possui hierarquia e nenhum estilo pré definido, mas tem sentido semântico de parágrafo.

Formas de uso

Agora já aprendemos a criar um bloco de texto simples, com título e parágrafo. Então vamos ao desenvolvimento: 

<h1>Sou um título</h1>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  Cras blandit elit eu nisl rutrum maximus. Sed suscipit elementum tortor. 
  Sed non mollis erat. "Sed vestibulum ornare risus a porta".
</p>

Usamos a tag de título para dar sentido ao trecho do parágrafo que vem logo em seguida. É importante ressaltar que é preciso inserir o texto dentro da tag <p>.

Resultado

Sou um título

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras blandit elit eu nisl rutrum maximus. Sed suscipit elementum tortor. Sed non mollis erat. "Sed vestibulum ornare risus a porta".

Mais recursos

Você pode precisar de mais recursos na construção de seu texto. Vamos abordar aqui mais tags HTML para marcação de texto.

<a></a>

Ela é utilizada para links externos e redirecionamento para outras páginas. Esta tag possui o atributo href que recebe o link e redireciona o usuário para o endereço fornecido. É comumente chamada de tag âncora e é um elemento de linha. 

 <a href="https://www.google.com.br" alt="descrição">Google</a>

<em></em>

A tag de ênfase (emphasis, em inglês). O efeito visual deste elemento de linha é um texto em itálico, mas a semântica, ou seja, o significado é de importância.

<p>Parágrafo com <em>texto relevante</em> junto com o conteúdo.<p>

Atrelado ao estilo itálico ainda temos outras tags, mas com sentido semântico diferente. Para deixar um texto em itálico apenas por estilo utilize a tag <i></i>, mas para citação de frase, nomes científicos ou de autores, utilize a tag <cite></cite>. Exemplo:

Eu acredito que às vezes são as pessoas que ninguém espera nada que fazem as coisas que ninguém consegue imaginar. - Turing, Alan

<strong></strong>

Vimos algumas formas de utilizarmos trechos de texto em itálico e os momentos mais adequados para cada um deles, mas agora chegou o momento do negrito. Para destacar a importância semântica relacionada ao negrito, utilizamos a tag <strong></strong>. 

<p>Este trecho está com <strong>destaque</strong></p>

Você também pode deixar o texto em negrito usando a tag <b>, mas o trecho envolvido por esta tag não terá valor semântico.

<u></u>

A tag <u> é a responsável pelo sublinhado e não possui valor semântico. 

<u>Este trecho está sublinhado</u>

Conclusão

Você aprendeu nesta aula sobre a importância da semântica nas diversas tags do HTML, e sobre estilização nas tags de marcação de texto.

Pratique ao final de cada teoria e crie pequenos projetos para treinar, afinal a prática te leva a entender melhor a utilidade e a sintaxe do HTML.
